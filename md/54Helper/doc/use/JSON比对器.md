# 一、简介
1、本工具可以对两个json串进行比对差异  
2、非json串会报“JSON不合法！”  
3、会显示两侧总共有多少处差异，并颜色高亮标识出差异处  

# 二、配置
## 2.1、右键菜单配置
在设置界面，** 配置右键菜单 **  
1、勾选“JSON比对器”  
2、右键菜单将显示“JSON比对器”  

![配置右键菜单-JSON比对器开启](../img/json-diff-1.png)
👆配置右键菜单-JSON比对器开启

## 2.2、popup菜单配置
在设置界面，** 配置功能菜单 **  
1、勾选“JSON比对器”  
2、popup菜单将出现“JSON比对器”  

![配置功能菜单-JSON比对器开启](../img/json-diff-2.png)
👆配置功能菜单-JSON比对器开启
