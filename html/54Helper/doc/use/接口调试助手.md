# 一、简介
1、`接口调试助手`是一个接口调试助手  
2、`接口调试助手`是一个基于postman的接口调试助手，使用方式同postman  

# 二、配置
## 2.1、popup菜单配置
在设置界面，** 配置功能菜单 **  
1、勾选“接口调试助手”  
2、popup菜单将出现“接口调试助手”  

![配置功能菜单-接口调试助手开启](../img/interface-postman-1.png)
👆配置功能菜单-接口调试助手开启

# 三、使用
## 3.1、开箱即用
1、点击“工具栏54Helper的popup”->“popup弹出”->“接口调试助手”->跳转至`接口调试助手`页面  

![接口调试助手的使用-popup菜单](../img/interface-postman-2.png)
